
# 安裝semantic-ui範例

## 說明

這個安裝範例是透過[bower](http://bower.io/)來安裝[Semantic-UI](http://semantic-ui.com/)。

## 執行步驟

安裝「Semantic UI」

``` sh
$ ./install.sh
```

利用「php-cli」啟動一個「Web Server」

``` sh
$ ./run.sh
```

觀看頁面

```
$ firefox http://localhost:8080
```
